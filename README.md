# Music Library

A simple web app that displays music tracks, supports search, and includes a working sort filter.

## Features

- Footer section included
- Sorting filter options:
  - Alphabetical A to Z
  - Alphabetical Z to A
  - Newest to oldest
  - Oldest to newest
- Search by song title or genre
- Fetches one track each for:
  - RnB
  - Afrobeats
  - Dancehall
  - Hip Hop

## Tech Used

- HTML
- CSS
- JavaScript (Vanilla JS)
- iTunes Search API

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
